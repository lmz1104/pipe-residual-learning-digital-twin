# Physics-Informed Residual Learning for Thermo-Fluid Pipeline Digital Twin

A research-oriented prototype for hybrid thermo-fluid pipeline modelling using mechanistic simulation and residual-learning correction.

This repository demonstrates a simplified **physics-informed machine learning workflow** for digital twin modelling of engineering pipeline systems.

---

# Scientific Motivation

Traditional thermo-fluid pipeline simulators are commonly constructed from simplified physical assumptions and empirical correlations.

Although mechanistic models provide strong physical interpretability, they may exhibit systematic prediction discrepancies under uncertain operating conditions due to:

- parameter uncertainty,
- unmodelled thermal losses,
- measurement noise,
- and simplified flow assumptions.

This project investigates whether machine-learning-based residual correction can compensate for structured model errors while preserving the physical consistency of mechanistic modelling.

The work focuses on a hybrid modelling strategy combining:

- physics-based thermo-fluid simulation,
- synthetic discrepancy generation,
- interpretable residual learning,
- and lightweight digital twin visualization.

---

# Research Objective

The objective of this repository is to construct a simplified physics-informed residual-learning framework for thermo-fluid pipeline systems.

The framework aims to:

1. preserve the interpretability of mechanistic modelling,
2. improve predictive accuracy through data-driven correction,
3. simulate realistic industrial uncertainty,
4. and demonstrate a lightweight digital twin prototype for engineering-oriented AI research.

---

# Methodology

The workflow consists of four major stages:

## 1. Mechanistic Thermo-Fluid Modelling

A simplified thermo-fluid pipeline model is constructed based on:

- conservation of mass,
- momentum balance,
- energy conservation,
- and Darcy–Weisbach pressure-drop relations.

The pressure-drop relationship is represented as:

\[
\Delta p = f\frac{L}{D}\frac{\rho u^2}{2}
\]

where:

- \(f\): Darcy friction factor
- \(L\): pipeline length
- \(D\): pipe diameter
- \(\rho\): fluid density
- \(u\): flow velocity

The mechanistic model provides physically interpretable baseline predictions for pressure and temperature evolution.

---

## 2. Synthetic Observation Generation

Pseudo-observed datasets are generated by introducing:

- structured discrepancies,
- random measurement noise,
- and systematic bias.

The goal is to emulate realistic industrial operating uncertainty while maintaining controllable experimental conditions.

This module enables:

- residual-learning experiments,
- robustness analysis,
- and hybrid-model evaluation.

---

## 3. Residual Learning Correction

Machine learning models are used to learn structured residual errors from the mechanistic baseline.

The corrected prediction is formulated as:

\[
y_{corrected} = y_{physics} + \hat r(x)
\]

where:

- \(y_{physics}\): mechanistic prediction
- \(\hat r(x)\): learned residual correction

Implemented residual-learning models include:

- Linear Regression (LR)
- Random Forest Regression (RF)

The residual-learning strategy improves predictive accuracy while preserving physical interpretability.

---

## 4. Digital Twin Demonstration

A lightweight digital twin prototype is constructed by integrating:

- thermo-fluid prediction,
- residual correction,
- vibration proxy monitoring,
- and visualization modules.

A simplified SDOF-based vibration surrogate model is included for coupled monitoring demonstration.
